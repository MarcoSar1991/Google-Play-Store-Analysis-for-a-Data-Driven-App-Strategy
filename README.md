# Google Play Store – Data-Driven Market Analysis

This project presents a data-driven analysis of the Google Play Store dataset to identify strategic opportunities for developing a sustainability-focused mobile application.

The analysis includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation and hypothesis testing
- Competitive landscape evaluation
- Sentiment analysis using user reviews
- Strategic proposal based on quantitative insights

## Project Objective

The goal of this project is to identify a promising app category based on:

- Market demand (Installs)
- Competitive intensity (Number of Apps)
- User perception (Sentiment Analysis)
- Business model dynamics (Free vs Paid)

The final outcome is a data-supported strategic proposal for a sustainability-oriented mobile application.
